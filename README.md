# My GULAG | Static Archive

# «Мой ГУЛАГ» | Статический архив

This repository contains a static mirror of the public website mygulag.ru.

В данном репозитории хранится статический снимок (зеркало) публичного сайта mygulag.ru.

---

## Project Context / Контекст проекта

EN: My GULAG is an oral history project of the GULAG History Museum. It collects video interviews and personal memories of people whose lives were affected by political repression and the Gulag.

RU: «Мой ГУЛАГ» — это проект устной истории Музея истории ГУЛАГа. Он собирает видеоинтервью и личные воспоминания людей, чьи жизни были затронуты политическими репрессиями и ГУЛАГом.
